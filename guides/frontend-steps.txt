[Vite / Vue]

Initialize
    - npm init vite {folder_name}
    - select vue, then javascript
    - cd {folder_name}
    - npm i && npm run dev

Install NPM Packages (npm i {package_one package_two ...})
    - pinia
    - vue-router
    - @vueuse/core
    - @vueuse/components
    - axios
    - lodash
    - @vitejs/plugin-vue
    - glob
    - vite-svg-loader

    (one-liner)
    - npm i pinia vue-router @vueuse/core @vueuse/components axios lodash @vitejs/plugin-vue glob vite-svg-loader

Install SASS
    - npm add -D sass
